## 画像の要件
- ファイル形式: JPG、PNG、WebP
- ファイルサイズ: 1MB以下推奨
- 解像度: 趣味画像は800x600px、プロフィールは400x400px推奨
- 著作権: 自分で撮影した写真または著作権フリーの画像を使用
